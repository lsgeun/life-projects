# 프로젝트 설명
- [file\_to\_folder\_and\_make\_properties](file_to_folder_and_make_properties.ipynb)는 ==(참고 자료의 )대표 파일==을 파일과 동일한 이름의 폴더를 생성한 후 그 폴더에 넣고, 파일에 properties 내용을 추가한다. 
- [end\_tag\_to\_properties](end_tag_to_properties.ipynb)는 빈 줄을 제외한 맨 마지막 줄에 위치한 태그를 찾아서, properties로 옮긴다.
- [end\_tag\_to\_properties 주피터 파일 실행으로 변경되지 않은 파일명](end_tag_to_properties%20주피터%20파일%20실행으로%20변경되지%20않은%20파일명.md)는 이름 그대로 [end\_tag\_to\_properties](end_tag_to_properties.ipynb)을 실행했을 때 변경되지 않은 파일명이다.
- [delete\_properties\_tag](delete_properties_tag.ipynb) properties에 있는 태그를 찾아서 지운다.
