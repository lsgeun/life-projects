
- 여러 잡다한 프로그램을 놔두는 폴더이다.

- [비밀번호 만들기](비밀번호%20만들기.py)는 아마도 숫자 4개, 영문자(대소문자) 4개, 특수문자 4개를 조합하여 비밀번호를 만든다.

- [오차 계산](오차%20계산.py)는 대학물리학실험 강의에서 사용한 파일인데 측정값을 리스트로 입력하면, 평균, 표준편차, 표준오차를 계산한다.

- [dutch\_pay](dutch_pay.py)는 더치페이한 내역을 입력하면, 자동으로 누가 누구에게 얼마를 줘야 하는지 계산한다.

- [time\_per\_distance](time_per_distance.ipynb)는 자전거나 달리기 등 속도가 나는 운동을 할 때, 각 평속마다 nkm를 가는데 걸리는 시간을 비교하기 쉽게 만든다.
